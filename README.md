# BrailleOCR

점역이 필요한 문서를 인식하는 한글OCR 시스템(프로토타입)  



글자의 위치를 추정하는 모델과 글자 분류모델을 나누어 학습하여  

한글문서 이미지를 입력으로 받아 점자로 번역한다  



### Requirements

| library       | version  |
| :------------ | :------- |
| pytorch       | 1.7.0    |
| opencv-python | 4.4.0.44 |
| numpy         | 1.18.5   |
| bs4           | 4.9.3    |
| requests      | 2.25.1   |
